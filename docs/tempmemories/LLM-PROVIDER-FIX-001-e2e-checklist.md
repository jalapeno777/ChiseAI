## Troubleshooting

### Exit Code 2: Security Failure

**Cause**: Demo mode not verified or position not flat

**Resolution**:
1. Verify `BYBIT_API_MODE=demo`
2. Verify `BYBIT_DEMO_API_KEY` is set
3. Ensure `BYBIT_API_KEY` (live) is NOT set
4. Check position was properly closed

### Exit Code 3: Credential Failure

**Cause**: Missing required credentials

**Resolution**:
1. Check `BYBIT_DEMO_API_KEY` and `BYBIT_DEMO_API_SECRET`
2. Verify `DISCORD_TRADING_WEBHOOK_URL` (optional but recommended)

### LLM Attempt Fails

**Expected**: With current credential issues, LLM providers will fail

**Behavior**: Test continues with fallback mode - this is expected and acceptable

**Evidence**: LLM failure is captured in evidence bundle with:
- Error message
- Latency measurement
- Fallback flag set

### Discord Notification Fails

**Cause**: Webhook not configured or network issue

**Behavior**: Test continues - Discord is not critical path

**Resolution**: Check `DISCORD_TRADING_WEBHOOK_URL` or skip if not needed
