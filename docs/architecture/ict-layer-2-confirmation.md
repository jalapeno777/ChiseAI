## Overview

Layer 2 implements the Signal Confirmation component using mathematical indicators and statistical validation. This layer confirms or rejects signals generated by the market structure context of Layer 1.

**Purpose:** Layer 2 answers the question "Is this signal statistically valid?" by applying quantitative filters and confirming Layer 1 observations with mathematical evidence.

**Critical Constraint:** Layer 2 must function independently as the baseline signal generator. All Layer 1 modifications are enhancements layered on top of Layer 2.

## Components

### 1. Technical Indicators

**Description:** Core mathematical indicators that generate directional signals.

**Indicator Set:**

| Indicator       | Signal Type | Parameters                            |
| --------------- | ----------- | ------------------------------------- |
| RSI             | Momentum    | period=14, oversold=30, overbought=70 |
| MACD            | Momentum    | fast=12, slow=26, signal=9            |
| Bollinger Bands | Volatility  | period=20, std_dev=2                  |
| ATR             | Volatility  | period=14                             |

**Confluence Scoring:**

Each indicator contributes to an overall confluence score:

```
confluence_score = Σ(indicator_signal × weight) / Σ(weights)
```

### 2. Statistical Validation

**Description:** Validates indicator signals against statistical significance to reduce false positives.

**Validation Methods:**

#### 2.1 Signal Coherence Check

Ensures multiple indicators agree on direction:

```python
def coherence_check(signals: list[IndicatorSignal]) -> tuple[bool, float]:
    """Check if signals are coherent (majority agreement)."""
    bullish = sum(1 for s in signals if s.signal == SignalDirection.BULLISH)
    bearish = sum(1 for s in signals if s.signal == SignalDirection.BEARISH)

    coherence = abs(bullish - bearish) / len(signals)
    is_valid = coherence >= COHERENCE_THRESHOLD

    return is_valid, coherence
```

**Threshold:** COHERENCE_THRESHOLD = 0.4 (40% majority)

#### 2.2 Historical Accuracy

Maintains per-indicator hit rate:

```python
@dataclass
class IndicatorStats:
    indicator: str
    total_signals: int
    successful_signals: int
    hit_rate: float  # successful_signals / total_signals
    avg_signal_strength: float
    last_updated: datetime
```

**Minimum Thresholds:**

- Minimum history: 50 signals
- Minimum hit rate: 0.48 (48%)
- Signals below threshold are downweighted

#### 2.3 Divergence Detection

Identifies when price and indicator diverge:

```python
def detect_divergence(price_data: list, indicator_data: list) -> DivergenceType:
    """Detect bullish or bearish divergence."""
    price_trend = calculate_slope(price_data[-20:])
    indicator_trend = calculate_slope(indicator_data[-20:])

    if price_trend > 0 and indicator_trend < 0:
        return DivergenceType.BEARISH_DIVERGENCE
    elif price_trend < 0 and indicator_trend > 0:
        return DivergenceType.BULLISH_DIVERGENCE
    return DivergenceType.NONE
```

### 3. Risk Metrics

**Description:** Quantifies risk parameters for each signal to ensure risk-adjusted positioning.

**Metrics Calculated:**

| Metric                | Calculation                  | Purpose                  |
| --------------------- | ---------------------------- | ------------------------ |
| ATR-based stop        | Entry × ATR × ATR_MULTIPLIER | Stop loss placement      |
| ATR-based take profit | Entry × ATR × TP_MULTIPLIER  | Take profit placement    |
| Risk/Reward ratio     | TP_distance / SL_distance    | Trade quality assessment |
| Position size         | Account_risk / SL_distance   | Kelly Criterion sizing   |
| Volatility factor     | Current_ATR / Historical_ATR | Market conditions        |

**Thresholds:**

```python
RISK_PARAMETERS = {
    "min_risk_reward": 1.5,       # Minimum 1.5:1 R:R
    "max_risk_reward": 5.0,       # Cap at 5:1 R:R
    "atr_multiplier_sl": 1.5,     # Stop at 1.5 × ATR
    "atr_multiplier_tp": 2.5,     # TP at 2.5 × ATR
    "max_position_pct": 0.02,     # Max 2% account risk
    "volatility_threshold": 2.0,  # Max 2× normal ATR
}
```

### Signal Confirmation Pipeline

```
│                                                                         │

│  │  Indicator   │    │   Statistical │    │     Risk      │            │
│  │  Signals     │───►│   Validation │───►│    Metrics   │            │
│  └──────────────┘    └──────────────┘    └──────────────┘            │
│         │                   │                   │                      │
│         │                   │                   │                      │
│         ▼                   ▼                   ▼                      │
│  ┌──────────────────────────────────────────────────────┐             │
│  │              Signal Confirmation Decision             │             │
│  │                                                       │             │
│  │  IF coherence >= 0.4                                  │             │
│  │  AND divergence == NONE or BULLISH (for long)        │             │
│  │  AND risk_reward >= 1.5                              │             │
│  │  AND volatility_factor <= 2.0                        │             │
│  │  THEN: CONFIRMED                                     │             │
│  │  ELSE: REJECTED                                      │             │
│  │                                                       │             │
│  └──────────────────────────────────────────────────────┘             │
│                              │                                          │
│                              ▼                                          │
│                    ┌──────────────────┐                                │
│                    │  Confirmed       │                                │
│                    │  Signal Output    │                                │
│                    └──────────────────┘                                │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### Confirmation Rules

| Condition                     | Action                  | Rationale                    |
| ----------------------------- | ----------------------- | ---------------------------- |
| Coherence >= 0.4              | Pass to validation      | Majority indicator agreement |
| Coherence < 0.4               | Flag for review         | Weak agreement               |
| Divergence detected (bearish) | Downgrade long signals  | Price/indicator disagree     |
| Divergence detected (bullish) | Downgrade short signals | Price/indicator disagree     |
| Risk/Reward < 1.5             | Reject signal           | Poor trade quality           |
| Risk/Reward >= 1.5            | Accept signal           | Adequate trade quality       |
| Volatility > 2× ATR           | Reduce position size    | Elevated risk condition      |

## Integration with Layer 1

Layer 2 receives Layer 1 context and applies modifiers:

```python
def apply_layer1_modifiers(
    signal: Layer2Signal,
    layer1_context: Layer1Context
) -> ModifiedSignal:
    """Apply Layer 1 structural modifiers to Layer 2 signal weights."""

    base_score = signal.confluence_score

    # Trend alignment modifier
    if layer1_context.trend_bullish and signal.direction == SignalDirection.BULLISH:
        modifier = +0.10
    elif layer1_context.trend_bearish and signal.direction == SignalDirection.BEARISH:
        modifier = +0.10
    elif layer1_context.trend_bullish and signal.direction == SignalDirection.BEARISH:
        modifier = -0.15
    elif layer1_context.trend_bearish and signal.direction == SignalDirection.BULLISH:
        modifier = -0.15
    else:
        modifier = 0.0

    # Zone proximity modifier
    active_zones = layer1_context.active_zones
    for zone in active_zones:
        if zone.contains(signal.entry_price):
            modifier += 0.05 if zone.type == signal.direction else -0.05

    # Structure strength modifier
    modifier *= layer1_context.structure_strength

    modified_score = base_score + modifier
    modified_score = max(0.0, min(1.0, modified_score))  # Clamp

    return ModifiedSignal(
        original=signal,
        modified_score=modified_score,
        total_modifier=modifier,
        layer1_context=layer1_context
    )
```

**Modifier Summary:**

| Layer 1 Condition                 | Signal Direction | Modifier |
| --------------------------------- | ---------------- | -------- |
| Trend aligns with signal          | Same direction   | +0.10    |
| Trend opposes signal              | Opposite         | -0.15    |
| Price in favorable zone           | Same direction   | +0.05    |
| Price in opposing zone            | Opposite         | -0.05    |
| Strong structure (strength > 0.7) | Any              | ×1.2     |
| Weak structure (strength < 0.3)   | Any              | ×0.8     |

## Backward Compatibility

**Critical Requirement:** Layer 2 operates identically when Layer 1 is disabled.

```python
def compute_signal(indicators, layer1_context=None):
    """Compute signal with optional Layer 1 context."""
    base_signal = compute_layer2_signal(indicators)

    if layer1_context is None or not ICT_CONFLUENCE_ENABLED:
        return base_signal  # Baseline behavior preserved

    return apply_layer1_modifiers(base_signal, layer1_context)
```

**Feature Flag Behavior:**

- `ICT_CONFLUENCE_ENABLED=true`: Layer 1 modifiers applied
- `ICT_CONFLUENCE_ENABLED=false`: Pure Layer 2 baseline
